Granulator

Designed and programmed a granular delay plugin in C++ using the JUCE framework.

Features

Processes a live input using granular synthesis with up to 60 concurrent active grains.
Users can control and randomize the following grain parameters:

  • Read position - Relative read position in the buffer
  • Random level range (dB)
  • Pitch +/- 1 octave
  • Duration between 16ms-900ms
  • Random panning range
  • The buffer can be frozen at any time.
