Unity Audio Toolkit

My Unity Audio Toolkit provides useful audio related functionality without the need to resort to middleware. It allows sound designers to easily create and control the behaviors of multiple audio events which can be called with simple code.

Features

Play, stop, pause or unpause audio event instances throughout the entire scene or at specific game objects.

Limit instances using the toolkit’s own priority and virtual voice system. It determines whether sounds are played or virtualized using a combination of distance scaled priority and instance limits (global, local and within radius).

Randomize and set audio parameters such as volume, pitch and filter cutoffs for sonic variety.

Control audio behaviors such as random or sequence audio clip selection, playback probability, and random positioning.
