Unity Audio Toolkit

I thought it would be fun to create a set of audio tools for Unity as an alternative to licensing middleware. It allows sound designers to easily create and control the behaviors of multiple audio events which can then be called via simple code.

Features

Play, stop, pause or unpause audio events throughout the entire scene or at specific game objects.

Limit instances using the toolkit’s own priority and virtual voice system. It determines whether sounds are played or virtualized using a combination of distance scaled priority and instance limit settings (global, local and within radius).

Add variation to sounds using randomized parameters and weighted random audio clip selection.

Control how audio behaves using playback probability, random positioning and repeating events.

Third person attenuation model that positions audio emitters so that they're panned according to the listener (ie. the camera) and attenuated based on distance from an attenuator position (ie. the player character).
